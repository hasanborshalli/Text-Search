This application allows users to search for specific text within a list of articles. It highlights the search term in both the title and content of each article, updating the number of matching articles found.
The project includes 3 index files (HTML for the structure, CSS for styling, JS for functionality of searching)

To run this application locally, follow these steps:

1-Clone the Repository: git clone https://github.com/hasanborshalli/Text-Search
2- navigate to the project directory
3- open index.html in a browser

To use the application:
In the search input field, type the text you want to search for. The application will automatically update and highlight matching terms in the articles as you type.
Also it will include the number of aricles found.
